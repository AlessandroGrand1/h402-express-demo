How It Works

  1. 1. Request

    Click “Read” and the server returns a 402 status with payment details.

  2. 2. Pay

    Your client "pays" the micro-fee on-chain (or via demo header) and retries the request automatically.

  3. 3. Receive

    The server verifies the payment and instantly serves the full, unlocked article HTML.

Powered by the H-402 Express middleware.
